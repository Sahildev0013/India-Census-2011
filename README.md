# India Census Data Analysis

## 📊 Project Overview

This project performs data analysis on the India Census dataset using Python and Pandas. It focuses on practicing data manipulation, filtering, grouping, and aggregation techniques.

## 🛠️ Tools & Technologies

* Python
* Pandas
* Jupyter Notebook / VS Code

## 📁 Dataset

The dataset contains information about:

* State name
* District name
* Population
* Male workers
* Female workers
* Religion-wise population
* Literacy data
* Households and more

## 🔍 Tasks Performed

* Hiding dataframe indexes
* Setting captions on dataframe
* Filtering districts
* State-wise population analysis
* Religion-wise aggregation
* Finding maximum and minimum values
* Sorting and ranking data
* Adding prefix and suffix to columns
* Setting columns as index
* Creating new calculated columns
* String-based filtering

## 📈 Example Questions Solved

* Total population state-wise
* Male workers in Maharashtra
* District with highest population
* Filter districts starting with letter A
* Top 5 states with highest population

## 🎯 Purpose

This repository is created for practicing Pandas dataframe operations and improving data analysis skills.

## ▶️ How to Run

1. Download the dataset
2. Open notebook in VS Code or Jupyter
3. Run cells step by step
